# DentalOffice - Система управления стоматологическим кабинетом

Приложение для автоматизации работы стоматологического кабинета, построенное на C# WinForms (.NET 8) с SQLite в качестве базы данных.

## Возможности

### Управление пациентами
- Создание, редактирование и удаление карточек пациентов
- Поиск по фамилии, имени и телефону
- Хранение медицинской информации: аллергии, страховой полис, заметки

### Расписание приемов
- Планирование приемов с указанием времени, врача и типа лечения
- Фильтрация по дате и статусу
- Статусы: Запланирован, В процессе, Завершен, Отменен, Не явился

### Зубная карта
- Визуальная карта зубов по нотации FDI (32 зуба)
- Цветовая кодировка состояния каждого зуба
- Состояния: Здоров, Кариес, Пломба, Коронка, Отсутствует, Имплант, Депульпирован, Нуждается в лечении

### Учет лечения
- Регистрация всех проведенных процедур
- Привязка к конкретному зубу и пациенту
- Типы: Осмотр, Чистка, Пломбирование, Удаление, Лечение каналов, Коронка, Мост, Имплантация, Отбеливание, Ортодонтия, Протезирование, Рентген

### Счета и оплата
- Создание счетов на основе проведенного лечения
- Частичная и полная оплата
- Способы оплаты: Наличные, Банковская карта, Перевод, Страховка
- Отслеживание неоплаченных счетов

### Панель управления (Dashboard)
- Общее количество пациентов
- Приемы на сегодня
- Доход за текущий месяц
- Сумма неоплаченных счетов

## Технологический стек

- **Платформа:** .NET 8
- **UI:** Windows Forms
- **База данных:** SQLite (через Microsoft.Data.Sqlite)
- **ORM:** Dapper (Micro-ORM)
- **Архитектура:** Service Layer + Models

## Сборка и запуск

### Требования
- .NET 8 SDK
- Windows (для WinForms)

### Запуск
```bash
cd DentalOffice
dotnet restore
dotnet build
dotnet run --project DentalOffice
```

### Публикация
```bash
dotnet publish DentalOffice -c Release -r win-x64 --self-contained true -o ./publish
```

## База данных

SQLite база данных автоматически создается при первом запуске в директории:
```
%LOCALAPPDATA%\DentalOffice\dental_office.db
```

Схема включает таблицы:
- `Patients` - пациенты
- `Appointments` - приемы
- `Treatments` - лечение
- `ToothRecords` - состояние зубов
- `Invoices` - счета
- `InvoiceItems` - позиции счета
